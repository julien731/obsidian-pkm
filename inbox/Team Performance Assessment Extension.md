
Extend the meeting self-assessment system to track team members' performance using the same framework.

## Idea

Apply the same meeting performance analysis I use for myself to assess team members:
- Track their contribution quality across meetings
- Identify patterns (positive and negative)
- Use data to drive coaching conversations
- Help them improve using the same benchmarks and anti-patterns

## Considerations

- **Privacy/consent**: Team members should know their performance is being tracked
- **Coaching framing**: Position as development tool, not surveillance
- **Same standards**: Apply the same criteria I hold myself to
- **Longitudinal value**: Patterns emerge over multiple meetings, not single snapshots
