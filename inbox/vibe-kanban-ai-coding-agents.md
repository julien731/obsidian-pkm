# Vibe Kanban — Orchestrate AI Coding Agents

**24.9k stars on GitHub** — [BloopAI/vibe-kanban](https://github.com/BloopAI/vibe-kanban)

## What it is

A project management tool built specifically for teams shipping with AI coding agents. The core workflow: **Plan → Prompt → Review**.

## The Problem It Solves

The bottleneck in AI-assisted development has shifted:

- **Old world:** Speed limited by how fast an engineer codes sequentially
- **New world:** Coding agents work in parallel on infinite tasks — but humans still need to plan and review that work. The bottleneck is now planning and review speed

## Key Features

- Run multiple coding agents **in parallel** to eliminate idle wait time
- Switch attention to whichever agent is ready for review
- Agent-agnostic — designed to work across different coding agents as the SOTA evolves
- End-to-end workflow for teams that ship with AI agents

## Why It's Relevant

Directly relevant to Nimble's AI adoption work and Julien's own AI-first development approach (budgeting SaaS, etc.). As coding agent usage scales across the team, the planning/review bottleneck becomes real — this addresses exactly that.
