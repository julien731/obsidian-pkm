Symphony is an open-source framework by OpenAI that transforms project management work into isolated, autonomous implementation runs handled by coding agents. It enables teams to oversee work at a higher level rather than supervising individual agents. Built primarily in Elixir and Python, it includes a specification for custom implementations in any language. Currently in early "engineering preview" status.

https://github.com/openai/symphony
