# Tanny to Become CSM


# Why

Tanny wants to move from her current position to a technical-oriented role, namely CSM.

# Expected Outcome

- Developed knowledge of foundational technical topics.
- Understands the basics of how to use and leverage raw data.
- Understands how and when to implement AppsFlyer's products.
- Knows how to use AppsFlyer and its dashboards to surface data.

# Training Plan

## Warm Up

- [x]  Read [https://www.marketingevolution.com/marketing-essentials/marketing-attribution](https://www.marketingevolution.com/marketing-essentials/marketing-attribution)
- [x]  Read [https://www.appsflyer.com/use-cases/cost-aggregation/cost-reporting-data-standardization/](https://www.appsflyer.com/use-cases/cost-aggregation/cost-reporting-data-standardization/)

## Technical Training

- [x]  Introduction to databases
    
    [Introduction to Relational Databases](https://mariadb.com/kb/en/introduction-to-relational-databases/)
    
- [x]  Introduction to SQL
    
    [SQL Tutorial](https://www.w3schools.com/sql/)
    
- [x]  Introduction to APIs
    - Types of APIs
    - Server, client
    - RESTful APIs
    - Authentication, authorization
    - Transactions (GET, PUT, POST, DELETE)
- [x]  Introduction to SDKs
- [ ]  Introduction to deeplinking (universal links)

## Data & Analytics
